Digital Twin for Construction Monitoring

Showcasing real-time 3D monitoring system using CesiumJS

Project Overview
Technologies Used CesiumJS, Cesium Ion, Google Sheets API, JavaScript, Speckle, Point Cloud

To address the challenge in tracking the construction progress of 47 buildings previously managed through static PDF reports, we delivered an interactive Digital Twin solution that offers real time visual monitoring and data insights.

We created a centralized Google Sheet, shared with all stakeholders, where the construction company updates the status of each building:

  • Building name
  • Number of floors completed
  • Floors currently in progress

Using CesiumJS, we developed a georeferenced 3D digital twin of the entire construction site. Each building floor (up to 14 floors) was modeled and uploaded to Cesium Ion. The system automatically fetches data from the Google Sheet and displays the 3D progress dynamically on the site map accurately showing completed floors on each building based on live updates.
